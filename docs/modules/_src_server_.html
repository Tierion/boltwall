Boltwall accepts a config object as an argument. With this configuration object, the server/api admin can setup custom caveats for restricting access to protected content For example, in this config, we have a time based caveat, where each satoshi of payment allows access for 1 second. caveatVerifier uses the available time based caveat verifier, however this can also be customized. getInvoiceDescription allows the admin to generate custom descriptions in the lightning invoice
